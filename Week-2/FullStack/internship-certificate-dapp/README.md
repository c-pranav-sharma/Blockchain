# Internship Certificate DApp

A Web3-themed React application for issuing and managing NFT-based internship certificates. This frontend connects to the `nft-record-api` backend.

## Features
- **Issue Certificates**: Form to record new NFT mints in the database.
- **Search & Filter**: Real-time search by Token ID or Owner Address.
- **Blockchain Links**: Direct links to Etherscan for transaction verification.
- **Responsive & Animated**: Built with Framer Motion and premium glassmorphism styling.

## Tech Stack
- **Frontend**: React, TypeScript, Vite
- **Styling**: Vanilla CSS (Premium Design System)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **API Client**: Axios

## Getting Started

### Prerequisites
1. Ensure the **NFT Record API** (backend) is running on `http://localhost:5002`.
2. Node.js installed.

### Installation
1. Navigate to the project folder:
   ```bash
   cd d:/BlockChain/Week-2/FullStack/internship-certificate-dapp
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the DApp
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open your browser to the URL provided (usually `http://localhost:5173`).
